## 📦 Proje Hakkında

Bu depo, **Ruti Medya** için hazırlanmış SMS gönderme aracının kaynak dosyalarını içerir.

Araç; farklı SMS servisleri ve sağlayıcıları üzerinden gönderim testleri yapmak, API bağlantılarını kontrol etmek, entegrasyon süreçlerini denemek ve geliştirici tarafında SMS servislerinin çalışma mantığını analiz etmek amacıyla geliştirilmiştir.

Sistem, **41+ farklı API desteği** ile çeşitli SMS servisleri üzerinde test ve doğrulama süreçlerinin daha esnek şekilde yürütülmesini sağlar.
